# GRI 업무 자동화 포털

경기연구원(GRI) 반복 행정 업무를 AI 기반으로 자동화하는 단일 페이지 웹 애플리케이션입니다.

---

## 서비스 목록

| 서비스 | 설명 | 상태 |
|---|---|---|
| 주간보고 취합 | 각 실별 주간보고 파일을 수집·병합·미리보기 | ✅ 구현 완료 |
| 채용검증 | 지원자 서류 자동 분류·검증·평가위원 매칭 | 🚧 PoC |

---

## 1. 주간보고 취합

### 기능 요약

- **파일 첨부** — 클릭 또는 드래그앤드롭으로 PDF·HWP·HWPX 업로드. 파일명에서 부서명 자동 매칭. 어떤 파일명이든 허용
- **파일 목록** — 업로드된 파일 칩 목록. 개별 다운로드·삭제. 파일 없으면 빈 상태 표시
- **부서 제출현황** — 미제출 부서 상단 정렬 리스트. 제출 시각·교체 시각(요일 포함) 표시. 진행률 바
- **자동 취합·병합** — 취합 버튼 클릭 시 PDF 병합(pdf-lib). 병합 완료 후 미리보기·다운로드 활성화
- **미리보기 팝업** — 95vw×90vh 모달. HWP 리본 편집기 UI 포함. 페이지 이동·줌 지원
- **다크모드** — 토글 버튼으로 전환

### 레이아웃

```
[Navbar]
[사이드바 260px] | [메인 컨텐츠]
  연도            |  [헤더]
  └ 월            |  ① 파일첨부 카드  |  ③ 부서제출현황 카드
    └ 차수         |     업로드존      |     진행률 + 리스트
                  |     파일목록      |
                  |  ② 취합버튼      |
```

### 파일 구조

```
index.html      — 단일 페이지 HTML
css/styles.css  — 전체 스타일 (CSS 변수 기반 디자인 토큰)
js/app.js       — 전체 비즈니스 로직 (모듈 패턴)
assets/         — 로고 등 정적 리소스
```

### 주요 모듈 (js/app.js)

| 모듈 | 역할 |
|---|---|
| `AppState` | 전역 상태 (파일 목록, 부서 목록, 세션, 지원자 등) |
| `Router` | 페이지 전환, 사이드바 표시 제어 |
| `UploadModule` | 파일 업로드·삭제·교체·렌더링 |
| `DeptModule` | 부서 추가·수정·삭제·제출현황 렌더링 |
| `MergerModule` | PDF 병합 (pdf-lib), 다운로드 |
| `PreviewModule` | PDF 미리보기 팝업 (PDF.js) |
| `SessionModule` | 세션 사이드바 렌더링, 연도·월 폴더 토글 |
| `DarkMode` | 다크모드 토글 |
| `RecruitModule` | 채용검증 기능 (하단 참조) |

## 2. 채용검증 (PoC)

### 배경

경기연구원 채용 프로세스 AI 자동화 가능성 검토 목적의 PoC.  
서류 검증·논문 검증·블라인드 채용 검토·평가위원 매칭 등 반복 업무 경감이 목표.

### 지원 파일 형식

PDF, HWPX, DOCX만 허용. 그 외 확장자 업로드 시 오류 팝업 표시.

### 파일명 규칙 (참고용)

```
YYMMDD_직책_모집분야_이름
예시: 250303_연구위원나급_교통계획및공학_홍길동
```

### 검증 항목

| # | 항목 | 내용 |
|---|---|---|
| 1 | 서류 검증 | 입사지원서·동의서·학력증명·경력증명·연구실적·자격증 누락/불일치 탐지 |
| 2 | 논문 검증 | 연구직 해당자 논문 실제 등재 여부·저자/공저 확인 (Google Scholar 기반) |
| 3 | 블라인드 검토 | 자기소개서 내 출신학교·지역·성별·가족관계 유추 문구 탐지 |
| 4 | 평가위원 매칭 | 지원자와 이해충돌 없는 평가위원 자동 추천 (동일 학교·지도교수·공저 제외) |

### 서류 카테고리 (6개)

| 카테고리 | 포함 서류 |
|---|---|
| 입사지원서 | 기본인적사항, 학력사항, 경력사항, 자격사항, 자기소개서 |
| 개인정보동의서 | 개인정보 수집·이용 동의서, 개인정보동의확인서 |
| 학력증명서 | 학위수여증명서, 성적증명서 |
| 경력증명서 | 재직증명서, 경력증명서 |
| 연구실적 | 연구실적목록, 논문 사본 (연구직 해당) |
| 자격증 | 자격증 사본 |

### UI 구조

```
[헤더: 채용검증 시스템 | Excel 다운로드]

  [초기 상태: 폴더 드래그&드롭 업로드 존]

  [업로드 후]
  ┌──────────────────────────────┬──────────────────────────┐
  │ 필터 탭 (전체 / 대기 / 완료) │                          │
  │ 폴더 업로드  Excel           │   상세 검증 결과          │
  │ ─────────────────────────── │   · 서류 항목 (아코디언)  │
  │ 지원자 목록 테이블            │   · 논문 검증             │
  │   · 서류/논문/블라인드 배지   │   · 블라인드 탐지 결과    │
  │   · 평가위원 평균 점수        │   · 평가위원 목록·점수    │
  │   · 상태 (이상탐지/대기/완료) │   · 검토완료 / 검토취소   │
  └──────────────────────────────┴──────────────────────────┘
```

### 필터 탭 기준

| 탭 | 포함 조건 |
|---|---|
| 전체 | 모든 지원자 |
| 대기 | 이상이 감지됐지만 아직 검토완료 처리 안 한 지원자 |
| 완료 | 담당자가 검토완료 처리한 지원자 |

### 검토 처리

- 상세 패널 우상단에 **검토완료** 버튼 표시
- 완료 상태에서는 **검토취소** 버튼으로 전환 (누르면 대기 상태로 복귀)
