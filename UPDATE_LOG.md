# GRI 채용자격검증 시스템 — 기능 업데이트 로그

**최종 업데이트:** 2026-06-19  
**프로젝트:** 경기연구원 채용자격검증 웹앱 (gri-weekly-report)  
**기술 스택:** Vanilla JS · HTML · CSS · Lucide Icons · SheetJS

---

## 1. 전체 구조

```
지원자검증 탭  ←──┐
                   ├── 채용 세션 사이드바 (GPT 스타일)
평가위원검증 탭 ←──┘
```

상단 탭 버튼 2개로 **지원자검증**과 **평가위원검증** 화면을 전환하며, 왼쪽에는 채용 세션 목록이 항상 표시됩니다.

---

## 2. 지원자검증 기능

### 2-1. 파일 업로드
- 드래그 앤 드롭 / 클릭 업로드 지원 (HWP, PDF, HWPX)
- 업로드 카드는 세로 가운데 정렬로 항상 고정

### 2-2. 대시보드 카드 (총 인원 / 서류 / 논문 / 블라인드)

| 카드 | 표시 형식 | 설명 |
|------|----------|------|
| 총 인원 | `N` | 전체 지원자 수 (변동 없음) |
| 서류 | `N명` | 서류 이슈 지원자 수 |
| 논문 | `N명` | 논문 이슈 지원자 수 |
| 블라인드 | `N명` | 블라인드 위반 탐지 지원자 수 |

- 서류·논문·블라인드 카드 클릭 시 해당 이슈 지원자만 목록 필터링
- 카드 재클릭 시 필터 해제
- `N명` 표기에서 숫자는 크게, `명`은 작고 흐리게 별도 스타일

### 2-3. 지원자 목록 테이블
- **6열 그리드:** 번호 / 이름+직책 / 서류(N/M) / 논문(N/M) / 블라인드(N건) / 상태
- **직책·분야 드롭다운 필터:** 선택 항목에 체크 아이콘(✓) 표시, 선택 시 파란색
- **탭 필터:** 전체 / 검토 (이상 탐지 건만)
- **상태:** 이상 없으면 `완료`, 미검토 이상 있으면 `검토`

### 2-4. 상세 패널
선택된 지원자 클릭 시 우측 상세 패널 오픈. **2개 탭** 구성:

**[검증 탭]**
- 서류 검증 목록 (항목별 O/X)
- 논문 등재 현황
- 블라인드 위반 항목 + **원본보기** 버튼
- **검토완료** / 완료해제 버튼

**[상세정보 탭]**
- 지원자 기본 정보 (소속, 직책, 학력, 전문분야 등)

### 2-5. 요약 메시지
- 상세 패널 상단에 위반 내용 요약
- 이상 없을 때: "전 항목 이상 없음, 검수 완료 권장"
- 이상 있을 때: **총 N건** 굵게 강조 + 항목별 줄바꿈 + 담당자 직접 검토 유도 문구

### 2-6. 원본보기 (HWP 뷰어)
- 블라인드 위반 항목의 원문 팝업 표시
- HWP 문서처럼 보이는 A4 스타일 페이지 (더미 데이터)
- 파일명 표시 바: `자기소개서_홍길동.hwp`
- 위반 문구 형광 하이라이트 처리
- 하단에 탐지 구간 요약 바

## 3. 평가위원검증 기능

### 3-1. 평가위원 테이블
- **6열 그리드:** 이름 / 소속 / 전문분야 / 연관성 지수 / 최고 / 최저
- 연관성 지수·최고·최저는 소수점 두 자리 표기 (예: `0.85`)
- 최고·최저 색상 구분 없이 블랙으로 통일

### 3-2. 평가위원 명단 업로드
- 툴바 내 업로드 라벨 클릭 또는 `.xlsx` / `.xls` / `.csv` 파일 선택
- SheetJS로 파싱, 이름(A열) · 소속(B열) · 전문분야(C열) 자동 인식
- 헤더 행 자동 스킵, 업로드 즉시 테이블에 반영
- 추가된 인원 수 `N명 추가됨` 상태 메시지 표시 (3초 후 사라짐)

### 3-3. Map 버튼
- 툴바 오른쪽에 위치
- 클릭 시 온톨로지 맵 팝업

---

## 4. 채용 세션 사이드바

- GPT 스타일의 좌측 세션 목록
- **새 채용** 버튼 클릭 시 세션 이름 입력 팝업 표시 → 이름 직접 지정 후 생성
- 각 세션에 마우스 호버 시 `···` 메뉴 버튼 표시
  - 이름 수정
  - 세션 고정 (고정 세션은 목록 최상단 유지)
  - 삭제
- localStorage에 세션 데이터 영속 저장

## 5. 온톨로지 맵

지원자 ↔ 분야 ↔ 심사위원 간의 연결 관계를 SVG 네트워크로 시각화.

| 노드 | 크기 | 색상 | 레이블 위치 |
|------|------|------|------------|
| 지원자 (8명) | 작은 점 (r=9) | 파란색 | 왼쪽 외부 |
| 분야 토픽 (6개) | 중간 (r=15) | 회색 | 아래 외부 |
| 심사위원 (4명) | 큰 원 (r=26) | 초록색 | 원 내부 |

- 연결은 불규칙 (지원자마다 1~3개 분야, 분야마다 1~2명 심사위원)
- 호버 시: 연결된 엣지만 강조, 무관한 노드는 흐리게 처리
- 하단 범례: 노드 크기 차이도 반영
